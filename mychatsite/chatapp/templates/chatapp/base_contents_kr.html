회사소개


아크위드는 고객과 거래처와 함께 지금보다 더 좋은 세상을 만들기 위해 노력합니다.


2016년 설립하여 연구 개발을 진행 중에 있으며, 주요 내용으로 사람의 판단, 선택 행위의 근간이 되는 지식 인지와 추론을 모방하여 컴퓨터가 지식을 축적, 처리, 활용하도록 하는 온톨로지 기반 지식정보 처리 응용 소프트웨어를 개발 중이고, 또한 사람의 지식정보 전달 및 습득에 중요한 수단인 인간의 언어를 컴퓨터가 이해하고 사람에게 반응 할 수 있도록 하는 자연어 처리와 이해 분야를 연구하며 관련 오픈 소스 응용 어플리케이션을 찾아서 재사용하고 새로운 어플리케이션 서비스로 구현 합니다.

서비스 주요 기술

사용자의 편의성을 증대에 필요한 부트스트랩 활용

부트스트랩 은 웹사이트를 쉽게 만들 수 있게 도와주는 HTML, CSS, JS 프레임워크이다.
하나의 CSS 로 휴대폰, 태블릿, 데스크탑까지 다양한 기기에서 작동한다.
다양한 기능을 제공하여 사용자가 쉽게 웹사이트를 제작, 유지, 보수 할 수 있도록 도와준다.

웹 서비스와 다양한 서비스 연계를 위해 Node.js와 Python Django 활용

Node.js는 확장성 있는 네트워크 애플리케이션 개발에 사용되는 소프트웨어 플랫폼이며,
Django는 파이썬으로 만들어진 무료 오픈소스 웹 애플리케이션 프레임워크(web application framework)로
쉽고 빠르게 웹사이트를 개발할 수 있도록 하는 다양한 구성요소로 이루어진 웹 프레임워크이다.

빅데이터 분석, 자연어 처리, 머신러닝, 딥러닝을 위한 Tensorflow

파이썬은 고급 프로그래밍 언어로, 플랫폼 독립적이며 인터프리터식, 객체지향적, 동적 타이핑 대화형 언어이며,
파이썬 기반 텐서플로우(tensorflow), 사이킷런(scikit-learn)과 같은 훌륭한 머신러닝 라이브러리를 활용,
빅데이터 분석, 자연어 이해, 처리 등 복잡하고 난해한 작업들을 쉽게 처리 할 있게 해준다.

아크위드 챗봇


서비스 소개

일반적으로 사람들이 기대하는 챗봇은 사람이 아무 말이나 걸면 거기에 부합하는 대답을 하기를 기대한다. 이러한 대화 방식을 사람 주도의 열린 대화라고 하며 대표적인 사례로 마이크로소프트의 죠(Zo)가 있다. 그러나 상용 비즈니스에는 이러한 방식이 적용되기 어렵다. 대부분의 기업은 비즈니스 목적상 대답 해야 할 답이 미리 준비 되어 있고 답변을 잘하기 위해 질문에 대한 이해도를 높이는 방법에 대해 많은 관심을 가지고 있다. 아크위드 챗봇은 이와 같은 요구에 부응하여 특정 영역의 대화 서비스를 위해 질문을 잘 이해하고 정확한 답변을 제공하기 위하여 시스템 주도의 닫힌대화 방식의 챗봇 서비스를 제공한다.
